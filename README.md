# Banking SQL Project

A beginner SQL project modelling a simple banking database.
Built to practise SQL fundamentals including table creation, data insertion, and querying.

## Database Structure
- **Customers** — customer personal details
- **Accounts** — bank accounts linked to customers
- **Transactions** — deposits, withdrawals and transfers
- **Branches** — bank branch locations across South Africa

## Skills Demonstrated
- Database design & schema creation
- Writing SQL queries (basic to intermediate)
- Data insertion and manipulation
- Joins, filtering, and aggregations

## Tools Used
- SQL (MySQL / SQLite)
